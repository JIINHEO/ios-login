# ios-login
firebase인증을 통해 로그인 기능을 구현한 iOS Application 

# 기능
- 이메일로 로그인, 비민번호 변경, 로그아웃
- Google로 로그인

# 활용기술 
- firebase
   > 모바일, 웹 애플리케이션 개발 플랫폼으로,
    백엔드 기능을 클라우드 서비스 형태로 제공하기 때문에 서버리스 애플리케이션 개발이 가능하다.
